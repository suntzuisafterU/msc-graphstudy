Detailed Description

Temporarily redirects std::cout, std::cerr and std::cin to / from memory buffers.

Warning
This does not affect data written via the I/O mechanisms provided by the C stdio.h facility (that is, writing to stdout using the fwrite function and its friends) or by the POXIX unistd.h facility (tat is, writing to STDOUT_FILENO using the write function and its friends). Data written to std::clog should also not be affected which is good because it can be used to report test status while I/O is captured.
